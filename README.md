### Aqua Bank — Personal Savings PWA

> iPhone-installable Progressive Web App to track a personal savings goal. Features PIN lock, 24-hour withdrawal delay with forced 30-word justification, monthly milestone tracker, and full offline support via service worker. Built to add real friction to impulsive withdrawals and keep long-term savings goals on track.

**Stack:** Vanilla JS · PWA · Service Worker · localStorage · Self-hosted on Oracle Cloud
